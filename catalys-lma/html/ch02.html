2.1. Introduction

The Catalys LMA application has traditionally been packaged as a zip file which is installed simply by unzipping the package in the desired destination directory. Starting with Catalys v2.2, we provide an automated platform-specific installer. The installer also helps you to:

  • Install and start the LMA as a service
  • Simplify the upgrade process

We continue to provide the zip package for customers who prefer to manually manage their installation process. However, for all other customers, we recommend using the installer, which is covered in the remainder of this section.
