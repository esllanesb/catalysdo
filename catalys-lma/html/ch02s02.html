2.2. Compatibility

Java Compatibility

Java 8 or later is required to run the LMA 2.2 installer and executables.

Catalys MIS Compatibility

The LMA interfaces with the Catalys Management Information Server (MIS). The MIS is available from Docs Portal.

Please consult the compatibility matrix for supported combinations of CameronFIX, Catalys Node, MIS and LMA.
